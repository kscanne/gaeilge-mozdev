Installation

Firefox Platform Installers

As of version 1.0.1 Firefox is available in Irish as the default language for Windows, Linux and Mac from the Firefox All Systems and Languages page.

To install Firefox download it to your PC or Mac then click on the installer file and simply follow the installation instructions given (in Irish).

Your feedback on our translation work so far would be greatly appreciated. Some good things to look for include:

  1. do the strings in the installer look like reasonable translations that would be well understood?
  2. do the labels for the primary toolbar buttons look like reasonable translations?
  3. do the items on the menubar and the items in the menus look like reasonable translations?
  4. do the items in the preferences/options window, primary tabs, look reasonable?

Please post to the mailing list if you have any bug reports.

Firefox XPI Instructions

For those who wish to use the language pack installer, instead of a full build:

  1. First you will need to install the latest Firefox release.
  2. Start up Firefox after installation, and find the XPI here (latest 1.0.4 localization).
  3. Scroll to tbe bottom of the page and select your platform, Linux, Mac or Windows.
  4. Scroll until you find the Irish XPI file "ga-IE.xpi" and click on it to install the language pack.
  5. Shut down Firefox.
  6. Restart Firefox. Shutdown again.
  7. Open up a command prompt/shell, go to the Firefox install directory and type:
    firefox -UILocale ga-IE
    Then hit [ENTER].

Step 6 may not be needed. If you are uncomfortable with using the command prompt you can install the Locale switcher extension...

Locale-Switcher Extension

The Locale switcher extension makes changing languages easier by listing your installed languages under your Tools menu. Good for folks working with the XPI version and still need English to fall back on.

Download this extension from: http://benjamin.smedbergs.us/switch-locales/
