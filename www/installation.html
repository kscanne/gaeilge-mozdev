INSTRUCTIONS (for Mozilla Firefox only)

  1. First you will need to install Firefox 1.0PR. You can grab it here:
    http://www.mozilla.org/products/firefox/releases/0.10.html#download
  2. Start up Firefox after install, and click on the link above. This will install the language pack. Shut down Firefox.
  3. Restart Firefox. Shutdown again.
  4. Open up a command prompt/shell, go to the Firefox install directory and type:
    firefox -UILocale ga-IE
    Then hit [ENTER].
Step 3 may not be needed. This is a work in progress, so you will still see some stings that are not translated.

Note there is an extension for changing locales in the UI, but it does not work with version 1.0PR, only in latest nightly builds.
