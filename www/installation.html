Irish installer files begin with firefox-1.0.ga-IE.

Some good things to look for:

  1. do the strings in the installer look like reasonable translations that would be well understood?
  2. do the labels for the primary toolbar buttons look like reasonable translations?
  3. do the items on the menubar and the items in the menus look like reasonable translations?
  4. do the items in the preferences/options window, primary tabs, look reasonable?

Please post to the list if you have any bug reports.

XPI INSTRUCTIONS (for Mozilla Firefox only)

For those who wish to use the language pack installer, instead of a full build.

  1. First you will need to install a Firefox 1.0 release. The latest is Release Candidate 2.
  2. Start up Firefox after install, and find the XPI here.
  3. Find firefox-1.0.ga-IE.langpack.xpi and click on it. This will install the language pack.
  4. Shut down Firefox.
  5. Restart Firefox. Shutdown again.
  6. Open up a command prompt/shell, go to the Firefox install directory and type:
    firefox -UILocale ga-IE
    Then hit [ENTER].

Step 5 may not be needed.
