Localized Firefox Addons

This page contains a list of Firefox addons which have been translated into Irish.

If you are aware of any Firefox extensions which have been translated into Irish but are not on our list please let us know. If you would like to translate Firefox extensions into Irish we suggest you pick any untranslated extension from the 'most popular' list to translate. The extension authors can assist you with any technical issues. Please then contact us when you have a translated extension you would like added to our list.
