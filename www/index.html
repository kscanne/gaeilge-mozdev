Firefox in Irish

This is the home page for the Irish localisation of the powerful Mozilla Firefox web browser. Coinciding with the release of version 1.0.1 Firefox is now available in Irish from the Firefox homepage for Windows, Mac and Linux.

And as of November 3rd 2004 the Irish locale files have been checked into mozilla.org's CVS which means that for every major release of Firefox, there will be an official release of Irish Firefox. This means Irish as default ... no more language switching necessary. It will be publicised and hosted on their servers.

So Download Firefox in Irish now and get surfing in your favourite language!

For those interested, you can view the status of the build (checkins, if it builds OK, ...) here:
http://tinderbox.mozilla.org/showbuilds.cgi?tree=Mozilla-l10n-ga-IE

Also, you can view and search files online:
http://lxr.mozilla.org/l10n-aviarybranch/

Example Irish file:
http://lxr.mozilla.org/l10n-aviarybranch/source/browser/locales/ga-IE/chrome/browser/aboutDialog.dtd You will notice some some characters are 'scrambled' and unreadable here. This is due to a conversion of certain characters (mainly characters with fadas) to unicode which is necessary for Firefox to correctly display it.
