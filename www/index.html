Firefox in Irish

This is the home page for the Irish localisation of the powerful Mozilla Firefox web browser. We are in the process of translating version 0.8 and hope to have a finished product in the next few weeks. In the meantime, you can download the browser for free (Windows, Mac, and Linux versions are available) and install the Irish language pack later by following the "Installation" link above.
